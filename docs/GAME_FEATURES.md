# 暗影深渊 — 功能说明文档

## 概述

**暗影深渊** 是一款 Godot 4.7 竖屏挂机 RPG（720×1280），中文暗黑奇幻主题。游戏采用单场景 + 底部五 Tab 导航，战斗在后台自动进行，玩家主要负责养成、装备、区域推进与 Meta 玩法。

- **入口场景**: `scenes/main.tscn`
- **核心 Autoload**: `GameManager`、`DataManager`、`SaveManager`、`AudioManager`

---

## 界面导航（5 个 Tab）

| Tab | 名称 | 功能 |
|-----|------|------|
| 0 | 战斗 | 自动挂机战斗、区域 Banner、Boss 挑战、区域切换、战斗日志 |
| 1 | 角色 | 装备、技能、天赋、宠物 |
| 2 | 冒险 | 路线图（10 区域）、深渊塔 |
| 3 | 工坊 | 锻造、强化 |
| 4 | 更多 | 任务、每日、商店、成就（含转生） |

## 战斗系统

### 波次挂机战斗（Tab 0）

- **自动战斗**：玩家与怪物按攻速计时自动攻防，无需手动操作
- **波次制**：每波 3~5 只怪物（随区域递增），依次从右侧入场
- **多目标**：玩家优先攻击最前方存活怪物；所有已激活怪物均可攻击玩家
- **波次奖励**：清完一波后弹出战利品对话框，汇总本波金币、经验、材料与装备
- **Boss 战**：点击「挑战 Boss」进入 1v1 特殊战斗，含狂暴机制；击败解锁下一区域

### 战斗参数

| 参数 | 默认值 |
|------|--------|
| 玩家攻速 | 1.2 秒/次 |
| 怪物攻速 | 1.5 秒/次（多怪时 ATK×0.7 平衡） |
| 装备掉率 | 15% |
| 材料掉率 | 30% |

### 死亡惩罚

- 损失 10% 金币
- 5 秒内攻击力 -20%（虚弱）
- 满血复活，重新开启波次

## 角色养成

### 属性

等级、攻击、防御、生命、暴击、暴击伤害、生命回复、吸血。由以下系统叠加：

1. 等级基础值
2. 6 槽装备
3. 强化（+1~+30，每级 +4% 属性）
4. 附魔
5. 天赋树
6. 宠物被动
7. 转生 / 称号加成

### 装备（6 槽）

武器、护甲、头盔、靴子、戒指、项链。品质：普通 → 神话（6 档）。战斗界面可见**武器 + 头盔 + 护甲**外观叠加。

### 技能（11 个主动）

物理 / 暗影 / 神圣三系，最多装备 4 个到技能栏，战斗中自动施法。含护盾、DoT、斩杀、复活等机制。

### 天赋（3 路线）

- **力量之路**：攻击、暴击、破甲、狂战
- **智慧之路**：经验/金币/掉落、CD 缩减、技能伤害
- **坚韧之路**：生命/防御/回复、格挡、吸血、不灭意志

### 宠物

收集、进化、设为主战。主战宠物提供属性加成并在战斗中造成周期伤害。

## 冒险

### 路线图（10 区域）

| 区域 | 等级范围 |
|------|----------|
| 黑暗森林 | 1–10 |
| 被遗忘的墓穴 | 10–20 |
| 影之领域 | 20–30 |
| 深渊裂隙 | 30–40 |
| 恶魔王座 | 40–50 |
| 混沌虚空 | 50–60 |
| 永夜之地 | 60–70 |
| 炼狱深渊 | 70–80 |
| 灭世之境 | 80–90 |
| 终焉之地 | 90–100 |

击败区域 Boss 解锁下一区域。

### 深渊塔

独立爬塔模式，每日 5 次，每层 3 波敌人，每 10 层 Boss，层数越高奖励越好。

---

## 工坊

- **锻造**：消耗材料 + 金币按配方生成装备
- **强化**：装备 +1~+30，成功率随等级递减
- **附魔**：随机附加词条

---

## 更多

- **任务**：击杀、升级、装备收集等目标
- **每日**：登录奖励、每日任务、经验/金币 Boost
- **商店**：宝石购买 Boost 与资源
- **成就 / 转生**：Lv.50+ 可转生获得永久加成

---

## 存档系统

- JSON 格式，每 30 秒自动保存
- 离线收益：按离线时长估算击杀与金币/经验
- 每日零点重置任务与部分统计

## 技术架构

```
GameManager (Autoload)
├── SkillSystem      自动施法、护盾、DoT
├── PetSystem        宠物战斗与掉落
├── TowerSystem      深渊塔
├── EnhanceSystem    强化附魔
├── AchievementSystem 成就转生
└── BattleWaveController  波次多怪战斗

main_ui.gd
├── BattleView       战斗首屏
├── PlayerVisual     装备外观叠加
├── BattleEnemyUnit  单怪 UI 单元
├── LootDialog       波次战利品弹框
└── PanelView        其他 Tab 动态列表
```
