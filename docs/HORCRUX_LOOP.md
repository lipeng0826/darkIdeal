# 界锚残片玩法设计 — 主线循环

> **灵感来源**：魂器式「大反派意志碎片化、诸界搜寻、集齐终局」  
> **本项目命名**：**界锚残片**（简称「残片」）— 不用「魂器」二字，避免与哈利波特直接重合，机制可对标。  
> **关联文档**：[万界裂隙世界观](MULTIVERSE_LORE.md) · [养成等级段](PROGRESSION_DESIGN.md)

---

## 一、设计目标

解决当前三个体验问题：

| 问题 | 残片主线如何解决 |
|------|------------------|
| 战斗像「框框干」 | 每场战斗服务于 **找线索 → 定位残片 → 击败守护者** |
| 系统多但无主线 | 全局唯一进度条：**已收集残片 X / 10** |
| 多元宇宙缺理由 | 每界 = 一枚残片藏身之处 + 一名可招募 **界域伙伴** |

**玩家心中始终只有一个问题**：「这一界的残片在哪？我还差几枚才能终结裂隙之主？」

---

## 二、世界观层（叙事）

### 2.1 反派：裂隙之主·虚无

- 万界裂隙不是自然灾害，而是 **裂隙之主** 将自身意志切成碎片、植入诸界的结果。
- 每一枚 **界锚残片** 既是它的一部分记忆，也是锁住该世界不被完全吞噬的「锚」。
- **悖论**：不摧毁残片，裂隙永不停止；摧毁残片，该界会失去最后的稳定——行者必须亲手做这件残酷的事。

### 2.2 玩家：裂隙行者

- 能感知残片波动，在诸界短暂锚定自身。
- 每收回一枚残片，裂隙之主在本界的投影减弱一分，但全局仇恨加深。
- 终局：集齐 **10 枚主残片** 后，在深渊枢纽面对裂隙之主真身。

### 2.3 小怪与 Boss 的新含义

| 角色 | 旧含义 | 新含义 |
|------|--------|--------|
| 普通怪物 | 挂机经验包 | 残片周围的 **记忆守卫**（被裂隙扭曲的本土生物） |
| 波次战斗 | 清图 | **探索推进**，掉落线索碎片 |
| 区域 Boss | 开门条件 | **残片守护者** — 守着该界残片的人/灵/概念 |
| 材料 | 养成资源 | 残片净化 / 线索合成所需 |

### 2.4 与伏地魔魂器结构的对应（仅供设计对照）

| 魂器叙事 | 本作映射 |
|----------|----------|
| 伏地魔分裂灵魂 | 裂隙之主分裂意志 |
| 各魂器有藏匿处与故事 | 每界 1 枚主残片 + 独立 lore |
| 摧毁魂器才能杀本体 | 集齐并净化主残片 → 解锁真 Boss |
| 魂器会反噬 | 残片持有期间，本界敌人略强化（可选机制） |
| 每个魂器一段冒险 | 每界独立探索 + 伙伴招募 |

## 三、核心玩法循环（单界）

```
┌─────────────────────────────────────────────────────────┐
│  坠入新界（穿越叙事）                                      │
└───────────────────────────┬─────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────┐
│  阶段 A：探索（挂机波次）                                  │
│  · 打记忆守卫 → 掉落「残片线索」                           │
│  · 本界迷雾/探索度上涨（UI 进度条）                        │
│  · 可遭遇本界商人、支线事件（后期）                        │
└───────────────────────────┬─────────────────────────────┘
                            ▼  线索 3/3
┌─────────────────────────────────────────────────────────┐
│  阶段 B：定位残片                                          │
│  · 线索合成 → 揭示残片名称与藏匿点                         │
│  · 战斗日志播放残片传说（lore）                            │
│  · 「挑战守护者」按钮解锁                                  │
└───────────────────────────┬─────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────┐
│  阶段 C：守护者战（Boss）                                │
│  · 击败区域 Boss → 获得「界锚残片」×1                      │
│  · 全局残片计数 +1                                        │
│  · 本界伙伴可招募（若未招募）                              │
└───────────────────────────┬─────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────┐
│  阶段 D：本界收尾                                          │
│  · 残片净化动画（短演出）                                  │
│  · 裂隙通道开启 → 下一界                                   │
│  · 本界探索状态归档（可回访，但不再重复主线）               │
└─────────────────────────────────────────────────────────┘
```

**单界目标时长**：15～30 分钟（含挂机），与等级段 10 级节奏大致对齐。

## 四、跨界保留 vs 本界重置（关键）

避免玩家觉得「每界白练」或「每界完全重复」。

### 4.1 永久保留（写入全局存档）

| 数据 | 说明 |
|------|------|
| 角色等级、经验 | 跨界成长 |
| 主装备、背包、金币、宝石 | 核心养成 |
| 已收集的 **界锚残片** 列表 | 主线进度 |
| 已解锁养成系统 | 见 PROGRESSION_DESIGN |
| **伙伴图鉴**（已遇见） | 见过即解锁图鉴 |
| 全局残片加成 | 每枚残片提供永久被动（见 §七） |

### 4.2 本界重置 / 本界独立（每进新界刷新）

| 数据 | 说明 |
|------|------|
| 本界线索进度 `clues: 0→3` | 重新收集 |
| 本界探索度 `explore_pct` | 从 0% 开始 |
| 本界活跃伙伴 | 仅在本界出战；离开后可「休眠」 |
| 本界商人库存 | 新界新货 |
| 本界支线事件触发标记 | 每界一次 |

### 4.3 伙伴规则（界域伙伴）

- 每界 **1 名主线伙伴**（与宠物系统复用或扩展 `realm_companions`）。
- **本界招募**：完成阶段 B（定位残片）或首次 Boss 前对话后入队。
- **跨界状态**：
  - **方案 A（推荐）**：伙伴跟随进入下一界，但等级按玩家等级的 80% 同步，避免某界伙伴过强。
  - **方案 B（更强重置感）**：伙伴休眠在图鉴，下一界招募新伙伴；旧伙伴可在「伙伴殿堂」消耗资源唤醒助战。
- 首版实现建议 **方案 A**，降低系统复杂度。

## 五、残片清单（10 界 + 3 隐藏）

### 5.1 主残片（每界 1 枚，Boss 掉落）

| 界 | Zone | 残片名 | 守护者（现有 Boss） | 残片传说（一句话） |
|----|------|--------|---------------------|-------------------|
| 0 | 暗影原界 | **枯根残念** | 森林之主·枯萎者 | 原界最后一颗树的年轮里，藏着裂隙最早的恐惧 |
| 1 | 汉末遗冢 | **将旗残念** | 墓穴之王·骸骨领主 | 未竟的天下霸业，被做成了一面永不倒下的将旗 |
| 2 | 丝路鬼市 | **镜花残念** | 影之女王·暗夜玫瑰 | 长安繁华倒影中，一朵永远不会醒来的梦 |
| 3 | 平行东瀛 | **番剧残念** | 深渊领主·炎魔贝利亚尔 | 某个世界「最终回」的胶片，卡在最后一帧 |
| 4 | 奥术王庭 | **法典残念** | 恶魔之王·路西法 | 禁忌魔法书里，被撕掉的那一页 |
| 5 | 机械坟场 | **协议残念** | 虚空之主·混沌 | 坠毁战舰 AI 的「消灭一切」指令，仍在循环 |
| 6 | 永夜北欧 | **月陨残念** | 永夜君王·暗月 | 诸神黄昏后坠落的月亮，还带着旧神的呼吸 |
| 7 | 青铜鬼城 | **祭鼎残念** | 炼狱之主·业火 | 殷商巫王献祭万人后，鼎里剩的一缕魂 |
| 8 | 像素次元 | **循环残念** | 灭世者·终焉 | 一段写坏的 `while(true)`，是整个游戏的噩梦 |
| 9 | 深渊枢纽 | **虚无残念** | 终焉之主·虚无 | 裂隙之主的心脏——集齐九枚后的终局之门 |

> **命名规则**：`{界域意象} + 残念`，统一后缀，方便 UI 与玩家记忆。

### 5.2 隐藏残片（可选收集，3 枚）

不挡主线，给肝帝与复盘玩家奖励。

| 隐藏残片 | 获取条件 | 加成 |
|----------|----------|------|
| **行者残念** | 单界探索度 100% | 经验 +3% |
| **盟约残念** | 本界伙伴满级/满好感 | 金币 +3% |
| **回溯残念** | 回访已通关界域击败「残影 Boss」 | 暴击 +1% |

每界最多触发 1 个隐藏条件，全收集隐藏 = 3 枚（非每界都有）。

### 5.3 终局条件

```
主残片 ≥ 9 枚（第 10 枚在深渊枢纽体内）
    ↓
解锁「深渊枢纽 · 裂隙真身」挑战
    ↓
击败终焉之主·虚无
    ↓
获得第 10 枚「虚无残念」→ 触发「重铸纪元天书」结局线
```

## 六、线索与探索（阶段 A/B 数值）

### 6.1 线索掉落

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 每波掉落线索概率 | 35% | 高于普通材料，让玩家感到在推进主线 |
| 单界所需线索 | 3 片 | 合成 1 次「残片定位」 |
| 保底 | 每 5 波必掉 1 线索 | 与装备保底类似 |
| 探索度 | 每波 +8%～12% | 12 波左右满探索（隐藏残片用） |

### 6.2 线索合成

```
残片线索 ×3 + 本界材料若干（见 multiverse_lore material_lore）
    → 触发「残片定位」事件
    → 解锁 Boss 挑战按钮
    → 播放该界残片传说（3～5 句）
```

### 6.3 UI 表现

- 战斗 Tab 顶部：**残片总进度** `界锚残片 3/10`
- 本界子进度：**线索 2/3** · **探索 64%**
- 世界地图节点：已通关界显示残片图标；未通关显示 `?`
- Boss 按钮：线索未满时灰色 + 提示「需要更多残片线索」

## 九、存档结构（规划）

```json
{
  "horcrux": {
    "collected_main": ["wither_root", "war_banner", ...],
    "collected_hidden": [],
    "total_main": 9,
    "final_unlocked": false
  },
  "realm_state": {
    "0": {
      "clues": 2,
      "explore_pct": 45,
      "fragment_located": false,
      "boss_cleared": false,
      "companion_recruited": false,
      "hidden_done": []
    }
  }
}
```

- `realm_state` 以 `zone_index` 为键；**切换界域时读写当前界**。
- 老存档迁移：已击杀 Boss 的界域视为 `boss_cleared=true` 且对应残片已收集。

## 十二、叙事节拍（裂隙之主存在感）

避免反派只在最后一关才出现。

| 时机 | 演出 |
|------|------|
| 新档开场 | 裂隙之主低语：「你找不到全部的……」 |
| 每获得 1 枚残片 | 低语反击：「这一枚，我会从记忆里夺回来」 |

## 十四、风险与原则

1. **线索要看得见** — 永远有进度条，不做纯解谜。  
2. **Boss 门槛要清晰** — 线索未满 = 按钮灰掉 + 说明，不让玩家懵。  
3. **残片加成要即时** — 拿到瞬间就加属性，最好弹「攻击 +2%」对比。  
4. **不抄袭名词** — 对外文案用「界锚残片」「裂隙之主」，不用伏地魔/魂器。  
5. **先 Zone 0 做透** — 第 1 界体验对了再铺 10 界。

---

## 十五、一句话总结

> **玩家是裂隙行者，在十个破碎世界里收集裂隙之主留下的界锚残片；每界重新探索、招募伙伴、击败守护者，集齐后方可终结万界裂隙。**

这是本作的主线，养成（装备/宝石/坐骑）是支撑主线的手段，而不是反过来。
